# Rainbow Text Generator

The Rainbow Text Generator is a text formatter, that get your text and transforms in Minecraft colour text (need to have a plugin for this).

![Result](https://i.imgur.com/LsA3ckH.png)

## Usage

Just download this package, then execute this command (in the project folder):

```bash
node .
```

Then choose what text you want, and the character before color code.

![Result](https://i.imgur.com/Rh4k4PI.png)
